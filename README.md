# Alura Space

## Descrição

Aplicação web utilizando o framework Django, simulando uma página que permite cadastro de usuário, login e acesso a imagens e informações de galáxias, nébulas e estrelas cadastras.


## Description

A web application using the Django framework, simulating a page that allows user registration, login, and access to images and information about registered galaxies, nebulae, and stars.
